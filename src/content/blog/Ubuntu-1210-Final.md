[Ubuntu 12.10 正式版](/blog/ubuntu-1210-final) [Quantal Quetzal](/blog/ubuntu-1210-final) 已经发布，目前已经可以下载。

[Ubuntu 12.10 正式版](/blog/ubuntu-1210-final)带来的更新有：

* 搭载[Linux](/tags/Linux) Kernel 3.5、GCC 4.7.2、Python 3.2。
* 移除 Unity 2D 模式，在不具备显卡加速的机器使用 LLVMpipe 利用 CPU 实现渲染。
* 统一化的 800M ISO 镜像，不再区分 desktop 和 alternative
* 增加对 OpenStack 架构即服务的支持。
* 在 Dash 中增加 Amazon 购物 Lens。
* 允许将 Web Apps 添加到 Dash 快速启动。
* 实现了 Juju 快速部署服务。
* 支持在通过 [Ubuntu](/tags/Ubuntu) Certification 的 UEFI 设备上引导。
